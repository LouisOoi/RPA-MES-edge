Bus Settings

Choose how this terminal talks to field devices. Everything else — points, scaling, rules — works the same regardless of transport.

Edge = Modbus TCP client. Each IO module is its own server with its own IP — set per device in the Devices tab. These are just defaults.

Logic Rules

Wire sensors to relays. IF a condition holds THEN drive an output. No code — pick from dropdowns.

ℹ️ v1: one condition per rule (IF → THEN/ELSE). Multi-condition (AND/OR), timers and counters are structured into the schema for a later release.
IF Oven Temperature > 180 °C for 2 s THEN set Maintenance LED = ON ELSE set Maintenance LED = OFF
IF Maintenance Button rising edge THEN pulse Maintenance LED 500 ms + log maintenance_request
IF Line Pressure > 8 bar THEN log machine_fault disabled
✓ Validation: no address conflicts, no dangling point references, no two rules driving the same output.
